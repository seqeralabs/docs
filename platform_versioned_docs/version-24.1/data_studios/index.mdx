## Prerequisites

### Conda

- Wave must be configured, generally `https://wave.seqera.io`

## Additional Conda packages

When you add a new Data Studio, you can customize its configuration to include a custom list of Conda packages.

```yaml
dependencies:
- ipython=7.13
- matplotlib=3.1
- pandas=1.0
- pip=20.0
- python=3.6
- scikit-learn=0.22
```
