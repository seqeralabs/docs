## Feature updates and improvements

- Updated the default Azure Batch Ubuntu image to SKU 22.04 to avoid issues with the existing image (SKU 20.04), which will be deprecated after April 23, 2025. For more information about the deprecation, see https://github.com/Azure/Batch/issues/174.
