## Feature updates and improvements

- **Fusion**
    - Added Fusion Snapshots support in AWS Batch compute environments, enabling Fusion to automatically restore jobs interrupted by AWS Spot instance reclamation. See [Fusion Snapshots](https://docs.seqera.io/fusion/guide/snapshots) for more information.

## Bug fixes

### Workflows

- Fixed container table pagination.
- Made task-level containers tab visible only when containers are available.

### Launch form

- Fixed a launch form issue with eager `parameter` keyword interpolation.
