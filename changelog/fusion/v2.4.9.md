## Feature updates and improvements

- Added support for authenticating with Azure Managed Identities.
- Added support for arbitrary fuse options as CLI args.

### How to pin this version

```bash
fusion {
  enabled = true
  containerConfigUrl = 'https://fusionfs.seqera.io/releases/v2.4.9-amd64.json'
}
```

:::info
For ARM64 CPU architecture, use `containerConfigUrl = 'https://fusionfs.seqera.io/releases/v2.4.9-arm64.json'`.
:::
