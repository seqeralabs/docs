### Feature updates and improvements

- Added license validation
- Integrated Fusion snapshots
- Made `stopFusion` timeout configurable (via `FUSION_STOP_SHUTDOWN_TIMEOUT`)

### Bug fixes

- Fusion shutdown only once
- File moves overwriting files scheduled for upload
- Forced unmount and upload dirty manager on failed unmount
- Corrected spot instance termination metadata path
- Errored when overwriting a symbolic link
- Added a timeout to Fusion stop function
- Uploaded symbolic links in parallel
