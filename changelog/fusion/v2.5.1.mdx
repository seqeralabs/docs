## How to pin this version

```bash
fusion {
  enabled = true
  containerConfigUrl = 'https://fusionfs.seqera.io/releases/v2.5.1-amd64.json'
}
```
_**NOTE**: For ARM64 use `https://fusionfs.seqera.io/releases/v2.5.1-arm64.json` url._
