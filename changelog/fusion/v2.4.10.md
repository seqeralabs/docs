## Bug fixes

- Ensure Azure client caches are accessed atomically.
- Endless upload logs when there is a concurrent write to the logs file.
- Corrupted data on a rename that has active uploads on target path.
- Do not upload `.fusion.symlinks` if there is no change.

### How to pin this version

```bash
fusion {
  enabled = true
  containerConfigUrl = 'https://fusionfs.seqera.io/releases/v2.4.10-amd64.json'
}
```

:::info
For ARM64 CPU architecture, use `containerConfigUrl = 'https://fusionfs.seqera.io/releases/v2.4.10-arm64.json'`.
:::
