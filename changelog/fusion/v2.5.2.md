## Feature updates and improvements

- Generate post-mortem pprof profile data [v2.5].
- Enable monitoring of goroutines and memory usage.
- Add goroutine ID to log messages.
- Support authenticating with Azure Managed Identities.
- Support arbitrary fuse options options as CLI args.

### How to pin this version

```bash
fusion {
  enabled = true
  containerConfigUrl = 'https://fusionfs.seqera.io/releases/v2.5.2-amd64.json'
}
```

:::info
For ARM64 CPU architecture, use `containerConfigUrl = 'https://fusionfs.seqera.io/releases/v2.5.2-arm64.json'`.
:::
