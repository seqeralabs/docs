## How to pin this version

```groovy
fusion {

  containerConfigUrl = 'https://fusionfs.seqera.io/releases/v2.4.6-amd64.json'
}
```

:::info
For ARM64 CPU architecture, use `containerConfigUrl = 'https://fusionfs.seqera.io/releases/v2.4.6-arm64.json'`.
:::
