## How to pin this version

```groovy
fusion {
  enabled = true
  containerConfigUrl = 'https://fusionfs.seqera.io/releases/v2.4.13-amd64.json'
}
```

:::info
For ARM64 CPU architecture, use `containerConfigUrl = 'https://fusionfs.seqera.io/releases/v2.4.13-arm64.json'`.
:::
