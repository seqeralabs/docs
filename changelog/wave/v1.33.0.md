## Feature updates and improvements

### Container building

- Added multi-platform container builds for amd64 and arm64 by @pditommaso in [#985](https://github.com/seqeralabs/wave/pull/985)

### Dependencies

- Externalized lib-pairing module to libseqera by @pditommaso in [#958](https://github.com/seqeralabs/wave/pull/958)

### General

- Added validation and documentation for commands in the container API by @munishchouhan in [#961](https://github.com/seqeralabs/wave/pull/961)
- Added support for cross AWS account IAM role authentication by @munishchouhan in [#979](https://github.com/seqeralabs/wave/pull/979)
- Added Server-Side Request Forgery (SSRF) protection to the validate-creds API by @munishchouhan in [#962](https://github.com/seqeralabs/wave/pull/962)

## Bug fixes

### General

- Fixed error messages by @munishchouhan in [#957](https://github.com/seqeralabs/wave/pull/957)
- Added explicit type casts for ECR auth token requests by @pditommaso in [#992](https://github.com/seqeralabs/wave/pull/992)
