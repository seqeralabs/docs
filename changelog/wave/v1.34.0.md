## Feature updates and improvements

### Container building

- Made `micromamba/v2` the default build template for Conda packages by @pditommaso in [#1024](https://github.com/seqeralabs/wave/pull/1024)

### Dependencies

- Bumped `io.netty:netty-bom` to `4.2.13.Final` by @cristianrcv in [#1053](https://github.com/seqeralabs/wave/pull/1053)

### General

- Made the server listening port configurable via the `WAVE_SERVER_PORT` environment variable by @bebosudo in [#1029](https://github.com/seqeralabs/wave/pull/1029)
- Made the STDOUT log pattern configurable via `WAVE_LOG_STDOUT_PATTERN` by @munishchouhan in [#1055](https://github.com/seqeralabs/wave/pull/1055)
- Removed `git tag --force` from the Wave release CI by @ShahzebMahmood in [#1056](https://github.com/seqeralabs/wave/pull/1056)
- Removed the legacy `wave/app` ECR push by @ShahzebMahmood in [#1057](https://github.com/seqeralabs/wave/pull/1057)
- Used an OIDC role for Maven publishing by @ShahzebMahmood in [#1059](https://github.com/seqeralabs/wave/pull/1059)
- Added Wave features documentation by @christopher-hakkaart in [#1040](https://github.com/seqeralabs/wave/pull/1040)
