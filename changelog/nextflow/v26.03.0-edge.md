## Feature updates and improvements

### Language features

- Added module system by @pditommaso in [#6650](https://github.com/nextflow-io/nextflow/pull/6650)
- Added multi-arch support to `arch` process directive by @pditommaso in [#6897](https://github.com/nextflow-io/nextflow/pull/6897)
- Added record types by @bentsherman in [#6679](https://github.com/nextflow-io/nextflow/pull/6679)
- Propagated errors from workflow outputs by @bentsherman in [#6876](https://github.com/nextflow-io/nextflow/pull/6876)

### Configuration

- Added `provider` option to Seqera executor config by @pditommaso in [#6908](https://github.com/nextflow-io/nextflow/pull/6908)

### Google Batch

- Added support for GCP Ops Agent by @ejseqera in [#6608](https://github.com/nextflow-io/nextflow/pull/6608)
- Added support for latest-generation Google Cloud machine families by @sosoihd in [#6841](https://github.com/nextflow-io/nextflow/pull/6841)

### Fusion

- Defaulted Fusion to v2.6 for Seqera executor by @pditommaso in [#6933](https://github.com/nextflow-io/nextflow/pull/6933)

### General

- Added NVMe disk allocation and `diskMountPath` support by @pditommaso in [#6879](https://github.com/nextflow-io/nextflow/pull/6879)
- Added `logStreamId` to Tower task record by @pditommaso in [#6877](https://github.com/nextflow-io/nextflow/pull/6877)
- Excluded container from task trace when containers are not enabled by @bentsherman in [#6824](https://github.com/nextflow-io/nextflow/pull/6824)
- Increased `tasks.cloudZone` width from 15 to 25 by @ramonamela in [#6907](https://github.com/nextflow-io/nextflow/pull/6907)

## Bug fixes

### Language features

- Fixed formatting bug with map key expression by @bentsherman in [#6893](https://github.com/nextflow-io/nextflow/pull/6893)
- Fixed path traversal, resource leak, and race condition in module system by @pditommaso in [#6929](https://github.com/nextflow-io/nextflow/pull/6929)
- Fixed remote modules when base dir is not the current working directory by @jorgee in [#6932](https://github.com/nextflow-io/nextflow/pull/6932)
- Fixed `stageAs` method signature in typed process by @bentsherman in [#6915](https://github.com/nextflow-io/nextflow/pull/6915)

### Configuration

- Fixed type detection of CLI params in v2 config parser by @bentsherman in [#6765](https://github.com/nextflow-io/nextflow/pull/6765)

### Google Batch

- Fixed Google Batch exit code when spot claim is successfully retried by @jorgee in [#6926](https://github.com/nextflow-io/nextflow/pull/6926)

### Kubernetes

- Fixed K8s token refresh by caching `K8sClient` at executor level by @pditommaso in [#6925](https://github.com/nextflow-io/nextflow/pull/6925)

### Plugins

- Fixed `HttpPluginRepository` empty plugin list check by @pditommaso in [#6650](https://github.com/nextflow-io/nextflow/pull/6650)

### General

- Fixed `TraceFileObserver` NPE when trace file already exists by @robsyme in [#6891](https://github.com/nextflow-io/nextflow/pull/6891)
