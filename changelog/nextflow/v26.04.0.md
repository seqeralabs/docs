## Feature updates and improvements

### Configuration

- Renamed `nextflow.preview.types` feature flag to `nextflow.enable.types` by @pditommaso in [#7067](https://github.com/nextflow-io/nextflow/pull/7067)
- Added support for validation of collection params with record types by @bentsherman in [#7025](https://github.com/nextflow-io/nextflow/pull/7025)

### General

- Added `-user-secret` and `-workspace-secret` flags to `nextflow launch` by @rnaidu-seqera in [#7040](https://github.com/nextflow-io/nextflow/pull/7040)

## Bug fixes

### Language features

- Fixed runtime type reflection in `nf-lang` by @bentsherman in [#7077](https://github.com/nextflow-io/nextflow/pull/7077)

### General

- Fixed ANSI colors in error messages by @ewels in [#7078](https://github.com/nextflow-io/nextflow/pull/7078)

## Dependencies

- Bumped `nf-tower@1.27.0`
