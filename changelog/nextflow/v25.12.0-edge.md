## Feature updates and improvements

### Language features

- Added `listDirectory()` to Path type and deprecate `listFiles()` by @bentsherman in [#6581](https://github.com/nextflow-io/nextflow/pull/6581)
- Set local task exit status when time limit is exceeded by @bentsherman in [#6592](https://github.com/nextflow-io/nextflow/pull/6592)

### Google Batch

- Added default `maxSpotAttempts` for Fusion Snapshots in Google Batch by @fntlnz in [#6652](https://github.com/nextflow-io/nextflow/pull/6652)
- Refactored Google Batch `getExitCode` to imperative style by @pditommaso in [#6649](https://github.com/nextflow-io/nextflow/pull/6649)

### Kubernetes

- Added `runtimeClassName` to the pod options by @alexpilotti in [#6633](https://github.com/nextflow-io/nextflow/pull/6633)

### Wave

- Added `wave.build.template` config option by @pditommaso in [#6639](https://github.com/nextflow-io/nextflow/pull/6639)

### Plugins

- Added README files for all plugins by @pditommaso in [#6660](https://github.com/nextflow-io/nextflow/pull/6660)
- Checked Nextflow version before loading plugins by @bentsherman in [#6591](https://github.com/nextflow-io/nextflow/pull/6591)
