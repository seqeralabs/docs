## Feature updates and improvements

### Language features

- Allowed boolean params to implicitly default to false by @bentsherman in [#6764](https://github.com/nextflow-io/nextflow/pull/6764)

### Azure Batch

- Handled Azure Batch `ActiveJobAndScheduleQuotaReached` with retry by @adamrtalbot in [#6874](https://github.com/nextflow-io/nextflow/pull/6874)
- Updated default Azure Batch VM image to Ubuntu 24.04 by @adamrtalbot in [#6844](https://github.com/nextflow-io/nextflow/pull/6844)

### Google Batch

- Reported actual GCP zone in Google Batch trace records by @pditommaso in [#6855](https://github.com/nextflow-io/nextflow/pull/6855)

### LSF

- Treated LSF job status `UNKWN` as `HOLD` by @bentsherman in [#5756](https://github.com/nextflow-io/nextflow/pull/5756)
- Used explicit memory units for LSF executor by @bentsherman in [#5217](https://github.com/nextflow-io/nextflow/pull/5217)

### General

- Added agent output mode (`NXF_AGENT_MODE`) by @edmundmiller in [eae9f7d](https://github.com/nextflow-io/nextflow/commit/eae9f7d09)
- Ensured main script is first in the `WorkflowRun` lineage record by @jorgee in [#6845](https://github.com/nextflow-io/nextflow/pull/6845)
- Implemented sched core (alpha1) by @pditommaso in [#6242](https://github.com/nextflow-io/nextflow/pull/6242)
