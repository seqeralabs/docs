## Feature updates and improvements

### Compute environments

- **Azure Batch support for Ubuntu 20.04 LTS ending**: The default Azure Batch Ubuntu image now uses sku 22.04. This ensures no issues with the existing Azure Batch Ubuntu image (sku 20.04) which will be deprecated after April 23, 2025. See, [https://github.com/Azure/Batch/issues/174](https://github.com/Azure/Batch/issues/174) for more information about the deprecation.

### Launchpad

- Implemented support for multiple extension dataset parameters in the input form.
