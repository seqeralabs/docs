[Organizations](https://docs.seqera.io/platform-cloud/orgs-and-teams/organizations) are the top-level structure in Seqera Platform, containing workspaces, members, and teams. Organization owners can add or remove members from an organization or workspace, and can allocate specific access roles within workspaces.
