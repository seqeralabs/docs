[Studios](https://docs.seqera.io/platform-cloud/studios) is a unified platform which hosts a combination of container images and compute environments for interactive analysis using preferred tools like JupyterLab and RStudio Notebooks, Visual Studio Code IDEs, or Xpra remote desktops. Each Studio session is an individual interactive environment that encapsulates the live environment for dynamic data analysis.
