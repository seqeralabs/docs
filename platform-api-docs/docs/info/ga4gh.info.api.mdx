The [GA4GH Workflow Execution Service (WES)](https://www.ga4gh.org/product/workflow-execution-service-wes/) is a protocol for running the same genomic data analysis in multiple cloud environments. Use GA4GH WES endpoints to send the same workflow to different compute centres and receive reliable, consistent results, enhancing the reproducibility and reusability of genomic analyses.
