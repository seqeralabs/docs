[Compute environments](https://docs.seqera.io/platform-cloud/compute-envs/overview) define the execution platform where a pipeline or [Studio](https://docs.seqera.io/platform-cloud/studios) will run. Compute environments enable users to launch pipelines and studios on a growing number of **cloud** and **on-premises** platforms.
