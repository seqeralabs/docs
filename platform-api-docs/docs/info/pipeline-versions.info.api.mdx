[Pipeline versions](https://docs.seqera.io/platform-cloud/pipelines/versioning) allow you to manage multiple versions of a pipeline configuration. Each version represents a distinct snapshot of the pipeline with its own parameters, configuration, and launch settings. You can create, update, and manage versions, including setting a default version for launches.
