# Seqera Platform v24.1 Release Notes

We're absolutely thrilled to announce the highly anticipated release of Seqera Platform version 24.1, which represents a significant milestone in our ongoing journey to deliver the most powerful, robust, and comprehensive platform for scientific computing teams. This release is packed with exciting new features, improvements, and optimizations that will fundamentally transform how you run your pipelines.

## What's New

In this release, we've worked tirelessly to enhance virtually every aspect of the Seqera Platform experience. Whether you're a seasoned bioinformatician or just getting started, version 24.1 has something for you. Let's dive into the highlights:

### Performance Improvements

We've made significant performance improvements across the board. Seqera Platform 24.1 leverages cutting-edge engineering to deliver blazing fast results, empowering you to seamlessly run even the largest workloads with ease. It's worth noting that these improvements are particularly noticeable in the following areas:

- **Run submission**: Run submission has been significantly improved.
- **Launchpad**: Launchpad performance has been enhanced.
- **Data Explorer**: Data Explorer is now faster than ever before.

### Enhanced Security

Security has always been a top priority for us, and version 24.1 takes our commitment to enterprise-grade security to the next level. We've introduced a variety of new security features that allow you to harness the full power of modern authentication and authorization paradigms. These include but are not limited to:

- Single sign-on integration with various identity providers
- Role-based access control with granular permissions
- Comprehensive audit logging for all workspace operations

### Developer Experience

Last but not least, we've made numerous improvements to the developer experience. The CLI has been streamlined, the API has been enhanced, and the documentation has been comprehensively updated. In today's rapidly evolving software landscape, developer productivity is paramount, and we believe these changes will empower developers to deliver better outcomes faster.

## Breaking Changes

It's important to note that this release contains a number of breaking changes that you should be aware of before upgrading. Please review the migration guide carefully.

## Conclusion

In conclusion, Seqera Platform 24.1 represents a quantum leap forward in scientific computing technology. We're excited to see what you'll build with these powerful new capabilities. As always, we'd love to hear your feedback, so don't hesitate to reach out!
